# Лабораторная работа №1: Нормализация данных в снежинку

## Описание
Трансформация данных о продажах товаров для домашних питомцев в аналитическую модель "снежинка".

## Структура проекта
- `source_data/` - 10 CSV файлов с исходными данными (по 1000 строк)
- `sql/DDL/` - скрипты создания таблиц
- `sql/DML/` - скрипты заполнения данными
- `sql/05_verify.sql` - проверочные запросы
